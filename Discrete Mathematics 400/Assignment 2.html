Problems

1. Relations and their properties [30 points] For the following, the relations are on ℤ (the integers).

  1. x|y: "x divides y". This holds when x is a factor of y, in other words there is a z such that y = x·z. Prove or show a counterexample for each of these propeties: reflexive, symmetric, transitive.
  2. x∤y: "x does not divide y". This holds when x is not a factor of y. Prove or show a counterexample for each of these propeties: reflexive, symmetric, transitive.
  3. x≡ny: "x is n-equivalent to y". This holds when x - y is a multiple of n. Prove or show a counterexample for each of these propeties: reflexive, symmetric, transitive.

2. Karatsuba multiplication [30 points] I recommend reviewing the lecture and the week 2 summary before tackling this problem.

  1. Using Karatsuba's algorithm, demonstrated in class, multiply x = 53 by y = 47. As in the class demonstration, write down the values of a, d, e, and x·y.
  2. Use the algorithm to find the product x·y where x = 4737 and y = 5345. To do this, divide each four digit number into two two-digit pieces. Computing a then requires finding xH·yH = 47·53. But you have already done that in the previous step so just use that result. The same goes for calculating d on the lower two digits of x and the lower two digits of y. As in the previous problem, write down the values of a, d, e, and x·y.

This shows that Karatsuba's algorithm can be extended to multiplying two numbers containing any number of digits by invoking it recursively. This way of defining an algorithm recursively will come up often in this and many other classes.

  2. Evaluate the polynomial for x = 11 and write down the value.
